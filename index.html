k8s-academic

Local Cloud-Native Architecture Simulation v1

⚙️ Tech Stack

  • Orchestration: Kubernetes (Minikube)
  • CI/CD: Jenkins
  • Containerization: Docker
  • Networking: NGINX Ingress & Cloudflare

🚀 CI/CD Workflow

  • Push to GitHub (nightcodex7/k8s-academic)
  • Jenkins Webhook Trigger
  • Docker Build & Push to Docker Hub
  • Automated Rolling Update via Kubectl

🌐 Architecture Details

Traffic routes from the custom domain through a Cloudflare Tunnel into the local network. It hits an NGINX Ingress Controller, which passes requests to a Kubernetes Service. The backend relies on a Deployment with an attached Horizontal Pod Autoscaler (HPA), dynamically adjusting pods from 1 to 6 based on CPU utilization metrics collected by the Metrics Server.
